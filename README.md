# Data Professional Survey Analysis — Power BI Dashboard

## Overview
This project analyzes survey responses from professionals working in data-related roles and presents the findings through an interactive Power BI dashboard. The analysis explores career paths, compensation, job satisfaction, technical preferences, and demographic patterns within the data field.

## Objective
To transform raw survey data into a structured, interactive dashboard that enables exploration of workforce trends, satisfaction metrics, and skill preferences among data professionals.

## Dataset Scope
The survey includes responses to questions covering:

**Role & Career Path**
- Current role title
- Whether the respondent switched into data from another career
- Difficulty of breaking into the data field
- Most important factor when seeking a new job

**Compensation & Industry**
- Current yearly salary (USD)
- Industry of employment

**Technical Preferences**
- Favorite programming language

**Job Satisfaction (1–10 Scale)**
- Satisfaction with salary
- Work/life balance
- Coworkers
- Management
- Upward mobility
- Learning opportunities

**Demographics**
- Gender
- Age
- Country
- Highest level of education
- Ethnicity

## Tools & Methods
- Power BI Desktop
- Data cleaning and transformation
- DAX measures and calculated fields
- Interactive visuals, slicers, and filters

## Dashboard Features
- Role and industry breakdowns
- Salary distribution and comparisons
- Satisfaction score analysis across multiple dimensions
- Programming language preference trends
- Demographic segmentation views
- Interactive filtering and drill-down capability

## Key Insights
- Satisfaction and salary patterns vary across roles and industries.
- Career switchers show different entry difficulty patterns.
- Programming language preferences differ by role group.
- Work/life balance and growth opportunity scores show measurable spread across demographics.
